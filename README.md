# 📊 Superstore Sales Analysis

## 📌 Project Overview

This project analyzes retail sales data to uncover business insights related to sales performance, profitability, customer purchasing patterns, and regional trends.

The project combines SQL for business analysis, Python for data processing and automation, Excel for reporting, and Power BI for interactive dashboard visualization to create an end-to-end analytics workflow.

---
## 🔧 Tools & Technologies

* Python (Pandas, NumPy, Matplotlib)
* SQL (MySQL)
* Power BI
* Excel
---

## 📊 Key Features

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Business analysis using SQL queries
* Data visualization using Python
* Automated report generation using Python (report.py)
* Interactive Power BI dashboard
* Exported Excel reporting for insights sharing
---

## 🗄️ SQL Analysis

Implemented SQL-based business analysis using:

Database creation and table design
Data retrieval using SELECT
Aggregation using SUM(), AVG()
Data grouping using GROUP BY
Sorting using ORDER BY

### SQL Queries Performed

* Total Sales Analysis
* Total Profit Analysis
* Monthly Sales Trend
* Category-wise Sales Analysis
* Sub-Category Performance Analysis
* Region-wise Profit Analysis
* Discount vs Profit Analysis
